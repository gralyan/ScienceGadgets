Constructors - Կառուցող

class Bicycle {

	double wheelDiameter; // in meters
	int sprocketTeethFront; // count
	int sprocketTeethRear; // count
	double candence = 0; // pedal revolutions per second
	double speed = 0; // in meters per second

	Bicycle(double wheelDiameter, int sprocketTeethFront, int sprocketTeethRear) {
		this.wheelDiameter = wheelDiameter;
		this.sprocketTeethFront = sprocketTeethFront;
		this.sprocketTeethRear = sprocketTeethRear;
	}
}

This constructor's parameters look similar to the uninitialized fields. These variables are different even though they are named similarly. In cases like this the local variable will always be used. In order to reference the field value when there is a similarly named local variable, you can use the keyword 'this' as seen above. This constructor simply sets the unknown variables to the values specified in the parameter.
